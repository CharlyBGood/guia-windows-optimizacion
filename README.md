# Guía de optimización y reparación de Windows

Tutorial interactivo de SinapsiaLab para limpiar, acelerar y mantener tu PC con Windows 10 / 11.

Sitio en producción: <https://tutoriales.sinapsialab.com/windows-repara/>

## Stack

- Vite + React + TypeScript
- Tailwind CSS
- Despliegue en Vercel

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

El build queda en `dist/` y se sirve bajo el `base` `/windows-repara/`.
